# Fitness Club Admin Backend

Серверная часть приложения для администратора фитнес-клуба.
Проект предназначен для управления клиентами, абонементами, тренерами, расписанием тренировок и базой данных клуба.

## Возможности

* Регистрация и авторизация администратора
* Управление клиентами
* Управление абонементами
* Работа с тренерами
* Расписание тренировок
* CRUD операции для базы данных
* REST API для взаимодействия с клиентским приложением

## Технологии
* Python/ FastApi
* Express
* PostgreSQL 
* JWT Authentication

## Структура базы данных

Основные таблицы:

* clients
* trainers
* memberships
* membership_types
* group_classes
* payments
* halls
* visits

Связи между таблицами реализованы через внешние ключи.

## Статус проекта

Проект находится в разработке.
